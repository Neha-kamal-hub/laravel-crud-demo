# Laravel Products CRUD Demo

A professional **Products Management Application** built with **Laravel** demonstrating full **CRUD** functionality with:

- Product listing
- Image upload and display
- Pagination
- Search functionality
- Fully responsive UI using Bootstrap 5
- Home page with project info

This project is developed for portfolio purposes to demonstrate my Laravel skills.
